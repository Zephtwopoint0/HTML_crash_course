Handy shortcuts and information

  • F2 - can be used to rename both opening and closing tags.
  • Shift + Alt + Up/Down - Will copy your current line or selected lines.
  • Alt + Up/Down - Will shift your current line or selected lines in the direction pushed.
  • Shift + Alt + f - Auto indents the current page.
  • Ctrl + d - Selects current word.
  • Ctrl + b - Opens or closes the sidebar.
  • ./ - to look in the same directory
  • ../ - to look in the directory above

Inline vs Block Elements

Block Elements

Block elements begin on a new line and will take up the entire width of the page.

Examples of block elements include: <div>, <h1> through <h6> & <p>

Inline elements

Inline elements only fill the space required and will begin on the same line.

Examples of inline elements include: <button>, <span> & <a>

Different tags used in the body

Headings, <h1> => <h6>

Block element: There will normally be a single <h1> on each page with the sub sections being denoted by <h2>, <h3> ect.

Paragraphs, <p>

Block element: Used to fill the main bodies of text on the webpage.

Div, <div>

Block element: Has no styling associated with it can be used to group elemtents together.

Navigation, <nav>

Block element: Used in the same was as a div but for the navigation bar (top of page).

Footer, <footer>

Block element: Used in the same was as a div but for the footer (bottom of page).

Horizontal line, <hr>

Block element: creates a horizontal line through the page.

Bold, <b>

Inline element: Makes text inside of the tags bold.

Italics, <i>

Inline element: Makes text inside of the tags styled with italics.

Span, <span>

Inline element: Makes text inside of the tags stylable to how you want.

Button, <button>

Inline element: Creates a button which can be interacted with.

Anchor, <a>

Discord server

Inline element: Used to add links to new pages, remember to add target="_blank" this will make the link open in a new tab.

Forms, <form>

Forms example

Inline element: Used to inconjunction with javascript to input information from the user. check the link to see further information.

Section, <section>

Block element: Used to seperate out each section of your main page.

Image, <img>

Block element: Allows you to add images by linking the source src="",
